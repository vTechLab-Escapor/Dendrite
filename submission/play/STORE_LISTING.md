## English

### Short description (max 80 chars)
```
Branch any reply into a mind-map. On-device AI, offline-ready. Your keys.
```

### Full description (max 4000 chars)
```
Think in branches, not lines.

Conventional AI chat is a straight line — every new reply pushes the last one
out of view, and exploring a tangent means abandoning your main thread.
Dendrite breaks the timeline open.

In Dendrite, a conversation is a TREE. Highlight any phrase in any message and
spawn a branch that explores it — without losing your place. Every thread of
thought lives on, navigable through a live mind-map of your whole conversation.

WHY DENDRITE
• Non-linear branching — select any text in any reply and branch off it. Each
  branch carries its selection as context, so the AI knows exactly what you're
  drilling into.
• Live mind-map canvas — your entire conversation as an interactive tree. Pan,
  zoom, tap a node to jump there. The active path is highlighted; bookmarks are
  starred.
• arXiv / GitHub analysis — paste a paper or repo link and Dendrite grows a
  multi-branch analysis tree (core idea, method, experiments, limitations…).
• On-device + cloud — run a local Qwen model for offline, private, zero-cost
  answers, or switch to the cloud per chat — even per branch.
• Bring your own key — connect NVIDIA NIM, OpenAI, Anthropic, Google Gemini and
  more. Your API keys are stored in the device's secure keystore, never on our
  servers (we don't run any).
• Local-first & private — all your history lives in an on-device database. We
  collect nothing.

Dendrite is named after the dendrites of a neuron: the branching structures
through which thought actually spreads.

One question. Many directions. Nothing lost.

Open source (MIT): https://github.com/vTechLab-Escapor/Dendrite
```

## 中文（简体）

### 简短说明（≤ 80 字符）
```
选中任意回复即可开分支，长成思维导图。端侧 AI，离线可用，自带密钥。
```

### 完整说明（≤ 4000 字符）
```
用分支思考，别走直线。

传统的 AI 聊天是一条直线——每条新回复都把上一条挤出视野，想顺着一个支线深挖，
就只能放弃主线。Dendrite 把这条时间线彻底打开。

在 Dendrite 里，一段对话是一棵「树」。选中任意消息里的任意一句话，就能就地生出
一条分支去探索它——而不丢失你原本的位置。每条思路都被保留下来，整段对话化作一张
实时的思维导图，随时漫游。

为什么选择 Dendrite
• 非线性分支——选中任意回复里的任意文字即可开分支。每条分支都把所选内容作为上下文
  带走，AI 因此清楚你在深挖哪一点。
• 实时思维导图——整段对话即一棵可交互的树。平移、缩放、点击节点跳转。当前路径高亮，
  收藏节点带星标。
• arXiv / GitHub 分析——粘贴论文或仓库链接，Dendrite 自动生长出多分支分析树
  （核心思想、方法、实验、局限……）。
• 端侧 + 云端——本地 Qwen 模型实现离线、私密、零成本作答；也可按对话、甚至按分支
  切换到云端。
• 自带密钥——接入 NVIDIA NIM、OpenAI、Anthropic、Google Gemini 等。你的 API 密钥
  存放在设备的安全存储里，绝不上传服务器（我们也不运行任何服务器）。
• 本地优先且私密——所有历史记录都存在设备本地数据库。我们不采集任何数据。

Dendrite 取名自神经元的树突：思想真正赖以蔓延的那些分叉结构。

一个问题。多个方向。无一遗漏。

开源（MIT）：https://github.com/vTechLab-Escapor/Dendrite
```
