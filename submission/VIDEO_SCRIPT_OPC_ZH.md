# Dendrite — OPC 大师创造营 参赛演示视频脚本（中文 · 闭环版）

**目标总时长：约 2:45**（硬上限 3:00）。
**语言：** 普通话旁白（NVIDIA Magpie TTS，voice = `Magpie-Multilingual.ZH-CN.Siwei`）+ 中文字幕。
**核心叙事（闭环）：**
> 真实任务 = 「把我的开源项目整理成一份参赛 PPT」。
> 用 **Dendrite** 把对项目仓库的探索散开成一棵知识树 → 一键 **导出 Markdown** → 喂给 **小浣熊（Little Raccoon）的 PPT 技能** → 自动生成参赛 PPT。
> 也就是说：**这份参赛作品的 PPT，本身就是用参赛作品 + 小浣熊一起做出来的。**

**与评分维度的对应：**
- **真实场景 & 闭环验证（35%）** → 场景就是「我现在要交的这份 PPT」；演示从探索一路跑到成品 PPT，端到端闭环、产物可见。
- **业务价值 & 可持续（30%）** → 把一次散乱的调研沉淀成结构化文档，可反复复用；本地优先 + 自带密钥，零额外成本长期运行。
- **工具整合 & 替换能力（25%）** → Dendrite（分支 / 思维导图 / 导出）+ 小浣熊（PPT 生成）多模块协同，且 Dendrite 引擎本身可在 NVIDIA / OpenAI / Anthropic / Gemini 间切换。
- **表达清晰（10%）** → 旁白按「痛点 → 方案 → 闭环 → 价值」推进，字幕全程跟随。
